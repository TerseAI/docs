Create a weekly digest of your pipeline in Attio and send it to Slack.

## Tutorial

<Steps>
  <Step title="Install the CLI">

```bash
npm install -g terse-cli
```

  </Step>
  <Step title="Scaffold a workflow project">

```bash
terse init acme-revenue --template weekly-pipeline-digest
cd acme-revenue
```

  </Step>
  <Step title="Test locally">

```bash
terse test --verbose
```

  </Step>
  <Step title="Deploy to Terse">

```bash
terse deploy
```

Navigate to the link provided to monitor your workflow.

  </Step>
</Steps>
