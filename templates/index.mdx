## How to use a template

<Steps>
  <Step title="Scaffold the starter">

```bash
terse init acme-revenue --template weekly-pipeline-digest
cd acme-revenue
```

  </Step>
  <Step title="Connect the required integrations">
    Open the integrations page with `terse integrate` and connect the systems the template expects.
  </Step>
  <Step title="Generate helpers">

```bash
terse generate
```

  </Step>
  <Step title="Test and deploy">

```bash
terse test --verbose
terse deploy
```

  </Step>
</Steps>
